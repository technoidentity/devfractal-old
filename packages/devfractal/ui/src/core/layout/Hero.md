### Default Hero

```jsx
<Hero>
  <HeroBody>always vertically centered</HeroBody>
</Hero>
```

### Hero with all properties

```jsx
<Hero variant="primary" size="medium" bold={false}>
  <HeroHead>
    always at the top(it can have any of the components like navbar,tabs etc...)
  </HeroHead>
  <HeroBody>
    always vertically centered(it can have any of the components like
    title,subtitle,content etc...)
  </HeroBody>
  <HeroFoot>
    always at the bottom(it can have any of the components like tabs etc...)
  </HeroFoot>
</Hero>
```
