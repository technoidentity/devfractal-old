### Default Content

```jsx
<Content>
  content class can be used in any context where you just want to (or can only)
  write some text. For example, it's used for the paragraph you're currently
  reading.
</Content>
```

### Content with all properties

```jsx
<Content size="medium">
  content class can be used in any context where you just want to (or can only)
  write some text. For example, it's used for the paragraph you're currently
  reading.
</Content>
```
